# 三、核心分析原则

## 1. 必须先识别联赛赛制

不同联赛的排名、积分动机和赛季阶段可能完全不同。

需要先判断本场属于哪一种赛事结构：

| 联赛形式 | 常见特点 | 分析重点 |
|---|---|---|
| 标准双循环联赛 | 每队主客场各交手一次 | 总轮次、主客场、争冠与保级分差 |
| 标准单循环联赛 | 每队只交手一次或赛程较短 | 主客场分配、赛程公平性、样本较小 |
| 三循环或多循环联赛 | 同一对手赛季交手三次以上 | 第三循环主客场分配、交锋熟悉度 |
| 常规阶段后分组 | 进入争冠组、欧战组或保级组 | 分组前积分、积分是否保留或折半 |
| 开幕季/闭幕季 | Apertura/Clausura分开计算 | 当前阶段排名、阶段冠军规则 |
| 常规赛加季后赛 | 常见于美职联、澳超等 | 常规排名、季后赛资格、主场优势 |
| 升降级附加结构 | 直接降级加附加赛 | 保级线、附加赛区、净胜球压力 |
| 春秋制联赛 | 主要在北欧、亚洲部分联赛 | 赛季阶段、天气、欧战资格赛冲突 |
| 跨年制联赛 | 英超、西甲等 | 冬歇、圣诞赛程、欧战与杯赛密度 |
| 分区或联盟制 | 东西区、南北区 | 分区排名、跨区对手强度差异 |
| 积分重置或折半 | 部分欧洲联赛分组后使用 | 常规赛排名价值和分组后真实分差 |

必须说明：

- 当前是第几轮
- 总共多少轮
- 是否进入第二阶段
- 积分是否重置、折半或保留
- 是否存在争冠组、保级组或季后赛
- 是否有直接降级和附加赛区
- 本场3分对双方实际价值

---

## 2. 积分榜不是单纯的实力榜

积分排名需要结合：

- 已赛场次
- 少赛或多赛
- 对手强弱
- 主客场数量
- 红牌和点球收益
- 绝杀和被绝杀
- xG积分
- 伤病周期
- 换帅前后表现
- 洲际赛事消耗
- 升班马开局红利
- 净胜球和相互战绩规则

必须判断：

- 谁的排名更真实
- 谁可能排名虚高
- 谁可能被积分低估
- 是否存在强队短期低位
- 是否存在弱队依靠赛程和运气高位
- 盘口是否已经修正这种错位

---

## 3. 赛季阶段决定比赛动机

### 赛季初

重点观察：

- 新教练和新援磨合
- 阵型变化
- 升班马适应能力
- 体能储备
- 样本量不足
- 盘口更依赖历史实力和阵容身价

### 赛季中段

重点观察：

- 长期实力是否稳定
- 伤病和阵容深度
- 洲际赛事与杯赛轮换
- 连续客场和旅行
- 换帅效应
- 状态回归

### 赛季冲刺阶段

重点观察：

- 争冠
- 欧战资格
- 升级
- 保级
- 附加赛资格
- 净胜球需求
- 无欲无求
- 主教练去留
- 合同和转会影响
- 年轻球员轮换
- 战意不对称

核心判断：

**必须分析每1分对双方的边际价值，而不是简单使用“强队战意高、弱队战意低”的模板。**

---

## 4. 必须区分近期状态与长期实力

建议同时使用：

| 时间窗口 | 主要用途 |
|---|---|
| 最近3场 | 短期状态、伤停和战术变化 |
| 最近5场 | 当前走势 |
| 最近8—10场 | 判断趋势稳定性 |
| 赛季整体 | 判断长期实力 |
| 主场/客场分项 | 判断真实主客差异 |

必须识别：

- 连胜是否来自弱队赛程
- 连败是否连续面对强队
- 大胜是否受红牌影响
- 进球是否依赖点球
- 零封是否依赖门将超常发挥
- 实际比分是否偏离xG
- 换帅后的提升是否已被盘口充分定价
- 是否存在状态回归风险

---

## 5. 主客场必须单独分析

不能只比较赛季总数据。

必须比较：

- 主队主场战绩
- 客队客场战绩
- 主场/客场场均进球
- 主场/客场场均失球
- 主场/客场xG和xGA
- 零封率
- 先失球率
- 上半场和下半场表现
- 主场球迷影响
- 客队旅行距离
- 草皮和气候适应
- 客队是否在客场主动降速
- 主队是否在深盘下效率下降

---

## 6. 必须校正对手强弱

最近5场不能只看胜平负。

建议把对手分为：

- 冠军级
- 上游
- 中游
- 下游
- 升班马
- 保级热门
- 杯赛轮换阵容
- 洲际赛事后疲劳阵容

示例：

> 主队最近5场取得4胜1负，但其中3场面对降级区球队，唯一面对上游球队时失利，因此不能直接评价为强势状态。

---

## 7. 强队更可能赢球，不等于当前让球值得支持

必须区分：

| 判断 | 含义 |
|---|---|
| 主队更可能取胜 | 90分钟胜率更高 |
| 主队让球有价值 | 当前盘口和价格合理 |
| 主队可能小胜 | 胜负方向正确，但可能无法穿盘 |
| 客队受让有价值 | 客队即使不胜，也可能守住盘口 |
| 强队不败但不宜追盘 | 概率方向与价格价值分离 |

常见风险：

- 欧战或杯赛前轮换
- 一周双赛
- 领先后降速
- 对手低位防守
- 主力前锋缺阵
- 赛季末只需要1分
- 初盘过深
- 升盘后热门过热
- 控球优势未转化为高质量机会

---

## 8. 盘口变化必须结合时间和消息

| 阶段 | 常见影响因素 |
|---|---|
| 开盘 | 基础实力、主客场和模型定价 |
| 早盘 | 专业资金、赛程和伤停预期 |
| 中盘 | 新闻、训练、市场热度 |
| 临场 | 首发、核心缺阵、天气、大众资金 |

必须分析：

- 升盘且降水
- 升盘但升水
- 降盘且热门低水
- 盘口不动但水位反转
- 欧赔与亚盘是否一致
- 大小球与胜负盘是否一致
- BTTS和大小球是否讲同一个故事
- 临场首发是否已经被市场提前消化
